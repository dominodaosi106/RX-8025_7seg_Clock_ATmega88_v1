# RX-8025_7seg_Clock_ATmega88_v1

# ATmega88P 7セグメントLED時計（RX-8025搭載）

## 概要
ATmega88Pマイコンを使用した6桁7セグメントLED時計です。  
高精度RTC「RX-8025」を搭載し、正確な時刻と年月日の管理を行います。

- 時刻表示：24時間表示 / 12時間表示（AM/PM付き）切替可能  
- 日付表示：YY.MM.DD形式  
- シンプルな2スイッチ操作  
- 毎正時（0時・12時）にブザー音  

### 開発環境
- マイコン：ATmega88P（1MHz内部発振）
- RTC：RX-8025（I2C接続）
- 言語：C言語（AVR-GCC）
- ライブラリ：カスタム `i2c.h` / `i2c.c`（ソフトウェアI2C）

## 主な機能

### 時刻表示
- 通常：HH:MM:SS（24時間または12時間表示）
- 24時間表示に切り替えた直後、2秒間「09:23:45」のように先頭に0を表示
- 12時間表示時はLED7（Gセグメント）が2秒間点灯
- コロン（:）は0.5秒間隔で点滅
- 毎秒にLED8（小数点）が約50ms点灯

### 日付表示
- S2短押しで「YY.MM.DD」形式を約2秒間表示
- 表示中は一部の小数点が消灯して視認性を向上

### 時刻設定
- S1 + S2 を2秒以上長押し → 時刻設定モード開始
- 設定項目：時 → 分 → 秒 → 保存

### 年月日設定
- 日付表示中に S2 を2秒以上長押し → 年月日設定モード開始
- 設定項目：年 → 月 → 日 → 保存
- 日付の無効値対策済み（例：2月30日、4月31日は自動で最大日に制限）

### ブザー
- 電源投入時：短いビープ音
- 毎正時（11:59:59 → 00:00:00 および 23:59:59 → 00:00:00）：短いビープ音

### LEDインジケータ
- **LED7（Gセグメント）**：12時間表示に切り替えた際に2秒間点灯
- **LED8（小数点）**：毎秒約50ms点灯（1秒の目安）

## 操作方法

### 通常モード

| 操作                     | 動作内容                                      |
|--------------------------|-----------------------------------------------|
| **S1 短押し**            | 24時間表示 ↔ 12時間表示（AM/PM）の切替        |
| **S2 短押し**            | 年月日表示を約2秒間表示                       |
| **S1 + S2 長押し（2秒）**| 時刻設定モード（時・分・秒）へ移行            |

### 年月日表示中

| 操作                     | 動作内容                                      |
|--------------------------|-----------------------------------------------|
| **S2 長押し（2秒）**     | 年月日設定モード（年・月・日）へ移行          |

### 設定モード共通

| 操作                     | 動作内容                                      |
|--------------------------|-----------------------------------------------|
| **S1 短押し**            | 次の設定項目へ移動（時→分→秒→保存 / 年→月→日→保存） |
| **S2 短押し**            | 設定値を +1                                   |
| **S2 長押し**            | 設定値を高速で増加（約10回/秒）               |

**設定の完了**：最後の項目（秒または日）で **S1を押す** と設定を保存して通常モードに戻ります。  
設定中は該当する桁が点滅します。

## ハードウェア構成

- **7セグメントLED**：6桁ダイナミック点灯（Timer2で約1kHz多重化）
- **スイッチ**：S1（PC2）、S2（PC3）
- **ブザー**：PD6（Timer0で約4kHz）
- **RTC割り込み**：INT0（PD2）で1Hz信号を受信
- **I2C**：ソフトウェアI2C（RX-8025接続）

**注意**  
7セグマスクとコロンマスクが2種類コメントアウトされているのは、異なる7セグメントホルダーに対応するためです。  
使用するホルダーに合わせて該当部分のコメントを切り替えてください。

## 注意事項

- 日付は月ごとの最大日数と簡易うるう年（4の倍数）を考慮して自動制限しています。
- 電池切れや電源喪失時はPONフラグが立ち、自動でクリアされますが、時刻がリセットされる可能性があります。
- 設定モード中に電源を落とすと、未保存の設定は失われます。
- 12時間表示時はAM/PMをLEDで表示します。

## 現在のバージョンでの改善点

- 日付の無効日対策済み（2/30など設定不可）
- 24時間表示切替時の視認性向上（先頭ゼロ表示）
- PONフラグの自動クリア対応
- スイッチ長押し時の自動インクリメント機能
